# 惡意軟體
- 病毒(virus)
```
> 可感染其他檔案
> 透過主動或被動傳播
> 感染主機數量隨時間增加
```
- 蠕蟲(Worm)
```
> 無法感染其他檔案
> 主動散佈自己
> 可以感染區域網路內具有相同弱點的主機
```
- 木馬(Trojan Horse)
```
> 無法感染其他檔案
> 只能「主動散佈自己」
> 一次只能感染一台主機
```

#

>  IOA(Indicator of attack) 入侵指標
```
Code Execution, Persistence, Stealth, Command Control
```
>  IOC(Indicator of compromise) 感染指標
```
Malware, Signatures, Exploit, Vulnerabilities, IP Address
```
